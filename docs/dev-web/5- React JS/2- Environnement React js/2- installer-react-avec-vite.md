#  Créer un projet React avec Vite

## Étapes :
1. Ouvrir le terminal
2. Lancer la commande :
```bash
npm create vite@latest mon-projet-react -- --template react
```
3. Aller dans le dossier :
```bash
cd mon-projet-react
```

1. Installer les dépendances:
```bash
npm install
```
2. Lancer le projet :
```bash
npm run dev
```
